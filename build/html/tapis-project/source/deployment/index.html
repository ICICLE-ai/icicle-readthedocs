Note

This guide is for users wanting to deploy Tapis software in their own datacenter. Researchers who simply want to make use of the Tapis APIs do not need to deploy any Tapis components and can ignore this guide.

The Deployment & Administration Guide for Tapis provides documentation for system operators wishing to run Tapis services in their own datacenter.

  • Considerations and Prerequisites: Covers considerations and prerequisites before embarking on a Tapis deployment.

  • Tapis Deployer: Covers using Tapis Deployer software to generate deployment scripts.

  • Tapis Vault: Describes HashiCorp Vault integration.

  • Managing Secrets with SkAdmin: Describes how to use SkAdmin to manage secrets.

  • Tapis Administration: Describes common administrative tasks that must be performed when operating Tapis.

  • Upgrading Tapis: Describes the process of upgrading an existing Tapis installation.

  • Tapis Artifacts & Architecture: Provides a detailed overview of the Tapis system architecture.
